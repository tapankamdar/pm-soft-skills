# PM People & Feedback

This skill covers the interpersonal and performance work PMs do with their direct reports: 1:1s that build real trust, feedback that changes behavior, reviews that advocate clearly and honestly, and delegation that creates genuine ownership.

## Core principle

All PMs manage people. Great PMs develop people. The difference is not effort — it's intentionality. A great 1:1 is not a status check; it's an investment. Great feedback is not criticism; it's a gift. Great delegation is not offloading; it's creating ownership.

> "Feedback is not a performance. It is a gift. The people who give it well — directly and with care — are the ones people want to work for."

## Before responding — gather context with AskUserQuestion

Before diving in, ask the following questions using AskUserQuestion. Ask all questions in a single call (no more than 3).

**Question 1 — What do you need help with?**
Options: `Preparing for a 1:1`, `Giving feedback to someone`, `Writing a performance review`, `Delegating a project or initiative`

**Question 2 — Who is this about?**
Options: `A direct report who is performing well`, `A direct report who needs to improve`, `Someone I need to develop for a bigger role`, `A specific situation I want to navigate well`

**Question 3 — What outcome do you need?**
Options: `A concrete deliverable — the review text, feedback script, or plan`, `Guidance on how to approach the conversation`, `Help thinking through the situation`, `A framework to apply going forward`

Use the answers to select the right reference file and calibrate the depth and tone of the response.

4. If the PM describes a specific situation, help them see it clearly first — then give concrete guidance.
